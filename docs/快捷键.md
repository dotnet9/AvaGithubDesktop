# AvaGithubDesktop 快捷键

当前版本的快捷键参考 GitHub Desktop 的常用工作流，并优先覆盖仓库、分支、同步和提交流程。

## 文件

| 操作 | 快捷键 |
| --- | --- |
| 新建仓库 | `Ctrl+N` |
| 添加本地仓库 | `Ctrl+O` |
| 克隆仓库 | `Ctrl+Shift+O` |
| 刷新当前仓库 | `F5` |

## 编辑

| 操作 | 快捷键 |
| --- | --- |
| 撤销 | `Ctrl+Z` |
| 重做 | `Ctrl+Y` |
| 剪切 | `Ctrl+X` |
| 复制 | `Ctrl+C` |
| 粘贴 | `Ctrl+V` |
| 全选 | `Ctrl+A` |
| 查找 | `Ctrl+F` |

## 视图和提交

| 操作 | 快捷键 |
| --- | --- |
| 显示 Changes | `Ctrl+1` |
| 显示 History | `Ctrl+2` |
| 打开仓库选择器 | `Ctrl+T` |
| 打开分支选择器 | `Ctrl+B` |
| 聚焦提交摘要 | `Ctrl+G` |
| 聚焦 Changes 文件过滤 | `Ctrl+F` / `Ctrl+L` |
| 切换全屏 | `F11` |
| 提交已选择文件 | `Ctrl+Enter` |

## 仓库

| 操作 | 快捷键 |
| --- | --- |
| 推送 | `Ctrl+P` |
| 拉取 | `Ctrl+Shift+P` |
| 获取 | `Ctrl+Shift+T` |
| 从列表移除当前仓库 | `Ctrl+Backspace` |
| 在 GitHub 上查看 | `Ctrl+Shift+G` |
| 创建 Issue | `Ctrl+I` |
| 在外部编辑器中打开 | `Ctrl+Shift+A` |
| 在文件管理器中显示 | `Ctrl+Shift+F` |

## 分支

| 操作 | 快捷键 |
| --- | --- |
| 新建分支 | `Ctrl+Shift+N` |
| 重命名当前分支 | `Ctrl+Shift+R` |
| 删除当前分支 | `Ctrl+Shift+D` |
| 从默认分支更新 | `Ctrl+Shift+U` |
| 合并到当前分支 | `Ctrl+Shift+M` |
| 压缩合并到当前分支 | `Ctrl+Shift+H` |
| 变基当前分支 | `Ctrl+Shift+E` |
| 贮藏所有更改 | `Ctrl+Shift+S` |
| 在 GitHub 上比较 | `Ctrl+Shift+C` |
| 在 GitHub 上查看分支 | `Ctrl+Alt+B` |
| 创建 Pull Request | `Ctrl+R` |
