# 更新日志

## 0.7.0.0 - 2026-05-26

### 新增

- ✅[新增]-Repository 菜单新增“在终端中打开”和“在文件管理器中显示”当前仓库操作，补齐 GitHub Desktop 常用仓库入口。

### 优化

- 🔧[优化]-新增 `RepositoryShellService` 统一维护 Windows、macOS、Linux 的外部进程启动逻辑，避免 ViewModel 混入平台细节。

## 0.6.0.0 - 2026-05-26

### 新增

- ✅[新增]-新增 Desktop 风格 Current repository 仓库选择器，支持弹出层过滤、最近仓库和分组仓库列表。
- ✅[新增]-新增仓库历史服务，自动记录用户打开过的本地仓库，并在开发环境扫描 `D:\github` 作为初始仓库列表。

### 优化

- 🔧[优化]-顶栏仓库入口由路径输入框调整为仓库选择按钮，操作方式更接近 GitHub Desktop。
- 🔧[优化]-仓库选择器样式封装到 `AvaGithubDesktop.Controls.Themes`，避免主应用入口承载样式细节。

### 测试

- 🧪[测试]-构建 `AvaGithubDesktop.slnx` 通过，完成资源文件、版本属性和仓库选择器弹出层验证。
